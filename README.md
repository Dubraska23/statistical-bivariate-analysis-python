# statistical-bivariate-analysis-python
Análisis estadístico bivariante e inferencial con Python aplicado a microdatos del CIS (SPSS). Implementación de limpieza de datos, pruebas de hipótesis (ANOVA, T-test, Tukey) y visualización avanzada con Seaborn para la extracción de insights sociodemográficos. Desarrollado en el Máster de Business Analytics de la UAM
